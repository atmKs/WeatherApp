# WeatherApp
Совместный проект - прогноз погоды

## WeatherApp
 UI + weatherapi. 
 Поиск прогноза погоды по городам (вывод погоды за 24ч + на след. 2 дня).
 Вывод погоды на основе геолокации устройства, с котрого идет запрос.
